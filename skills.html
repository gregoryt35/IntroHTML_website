Something I can teach someone is how to write a simple Java program!

Any Java program requires three things:

  • At least one class
  • A main method
  • At least one statement

Using these, we can write our own Java program.

Step 1: Declare a class by writing the keywords "public class" followed by a class name. For example: public class Main

Step 2: Write a main method within the class. The main method is always written like this: public static void main(String[] args)

Step 3: This is where you can use your imagination. You can write an entire algorithm or keep it simple with a print statement. For example: System.out.println("Hello World"); (Make sure to add a semicolon at the end of every statement!)
